# 基本命令

* `git init`	初始化

* `git add -A`   添加
* `git commit -m "This is a description!"`   提交
* `git status ` 仓库状态
* `git diff`   查看变化
* `git log/reflog`  查看版本
* `git reset --hard HEAD`  回退上一个版本   id任意版本
* 工作区
* 版本库
  * 暂存区 & 分支
* `git checkout --readme.txt`  撤销工作区修改
* `git reset HEAD <file>`  撤销暂存区修改
